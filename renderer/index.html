NetRecon

Professional attack surface mapper and vulnerability correlator. Designed for penetration testers, red teams, and security engineers.

Capabilities

  • Concurrent TCP port scanning with per-service protocol probes
  • Version fingerprinting via banner grabbing (SSH, HTTP, Redis, MySQL, MongoDB…)
  • CVE correlation against NIST NVD — mapped to discovered versions
  • Misconfiguration detection (unauthenticated Redis, exposed Docker, EternalBlue…)
  • Passive OSINT: Certificate Transparency subdomain enum, DNS/DoH, ASN lookup
  • Export to JSON (SIEM-ready), HTML (client reports), CSV
